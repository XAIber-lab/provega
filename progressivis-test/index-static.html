Progressive Vega-Lite + ProgressiVis

NYC taxi pickups (Uber 2014 sample) — static progressive playback

This variant uses the local CSV data/uber-rides-aug14-sample.csv (no WebSocket). ProVega progressively feeds the dataset according to the chunking config, so the progress bar, ETA, and quality panels reflect completion.

Serve statically (e.g., python -m http.server 8000) and open this file. Adjust chunk_size or total_chunks below if you change the sample size.
